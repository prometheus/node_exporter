# processes collector

The processes collector exposes metrics about processes.

## Metrics

| Metric | Description | Labels |
| --- | --- | --- |
| node_max_processes | Number of max PIDs limit | n/a |
| node_max_threads | Limit of threads in the system | n/a |
| node_pids | Number of PIDs | n/a |
| node_state | Number of processes in each state. | state |
| node_threads | Allocated threads in system | n/a |
| node_threads_state | Number of threads in each state. | thread_state |
