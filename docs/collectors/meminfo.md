# meminfo collector

The meminfo collector exposes metrics about meminfo.

## Supported Platforms

- Linux

## Data Sources

- /proc/meminfo

## Metrics

| Metric | Description | Labels |
| --- | --- | --- |
| node_memory_Active_bytes | Memory information field Active_bytes. | n/a |
| node_memory_MemFree_bytes | Memory information field MemFree_bytes. | n/a |
| node_memory_MemTotal_bytes | Memory information field MemTotal_bytes. | n/a |
