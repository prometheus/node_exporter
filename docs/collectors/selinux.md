# selinux collector

The selinux collector exposes metrics about selinux.

## Metrics

| Metric | Description | Labels |
| --- | --- | --- |
| node_selinux_config_mode | Configured SELinux enforcement mode | n/a |
| node_selinux_current_mode | Current SELinux enforcement mode | n/a |
| node_selinux_enabled | SELinux is enabled, 1 is true, 0 is false | n/a |
