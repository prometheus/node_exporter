# paths collector

The paths collector exposes metrics about paths.

## Configuration Flags

| Flag | Description | Default |
| --- | --- | --- |
| path.procfs | procfs mountpoint. |  |
| path.rootfs | rootfs mountpoint. | / |
| path.sysfs | sysfs mountpoint. | /sys |
| path.udev.data | udev data path. | /run/udev/data |
