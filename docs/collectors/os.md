# os collector

The os collector exposes metrics about os.

## Metrics

| Metric | Description | Labels |
| --- | --- | --- |
| node_os_support_end_timestamp_seconds | Metric containing the end-of-life date timestamp of the OS. | n/a |
| node_os_version | Metric containing the major.minor part of the OS version. | id, id_like, name |
