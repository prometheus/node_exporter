# netinterface collector

The netinterface collector exposes metrics about netinterface.

## Metrics

| Metric | Description | Labels |
| --- | --- | --- |
| node_netinterface_collisions_total | Total number of CSMA collisions on the interface. | n/a |
| node_netinterface_receive_bytes_total | Total number of bytes received on the interface. | n/a |
| node_netinterface_receive_packets_total | Total number of packets received on the interface. | n/a |
| node_netinterface_transmit_bytes_total | Total number of bytes transmitted on the interface. | n/a |
| node_netinterface_transmit_packets_total | Total number of packets transmitted on the interface. | n/a |
