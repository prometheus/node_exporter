# swap collector

The swap collector exposes metrics about swap.

## Metrics

| Metric | Description | Labels |
| --- | --- | --- |
| node_swap_priority | Swap device priority. | n/a |
| node_swap_size_bytes | Swap device size in bytes. | device, swap_type |
| node_swap_used_bytes | Swap device used in bytes. | n/a |
