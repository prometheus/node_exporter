# pressure collector

The pressure collector exposes metrics about pressure.

## Metrics

| Metric | Description | Labels |
| --- | --- | --- |
| node_pressure_cpu_waiting_seconds_total | Total time in seconds that processes have waited for CPU time | n/a |
| node_pressure_io_stalled_seconds_total | Total time in seconds no process could make progress due to IO congestion | n/a |
| node_pressure_io_waiting_seconds_total | Total time in seconds that processes have waited due to IO congestion | n/a |
| node_pressure_irq_stalled_seconds_total | Total time in seconds no process could make progress due to IRQ congestion | n/a |
| node_pressure_memory_stalled_seconds_total | Total time in seconds no process could make progress due to memory congestion | n/a |
| node_pressure_memory_waiting_seconds_total | Total time in seconds that processes have waited for memory | n/a |
